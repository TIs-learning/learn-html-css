Belajar Membuat tabel

Tabel standard

Perkenalan tag untuk membuat table: tr untuk membuat row / baris. th untuk membuat kolom header table. td untuk membuat kolom isi table. caption untuk keterangan table. thead untuk grup header table. tbody untuk grup isi table. tfoot untuk grup footer table.
